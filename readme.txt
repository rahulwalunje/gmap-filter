== Description ==
= Gmap Filter =
This plugin provides a simple way to display address on Google map, which plots entered address on Gmap and filter according to groups.

[See live demo](http://www.wpshopee.com/gmap-viewer/)

== Installation ==
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Gmap Filter** and click \"*Install now*\"
1. Alternatively, download the plugin and upload the contents of 'gmap-filter.zip' to your plugins directory, which usually is '/wp-content/plugins/'.
1. Activate the plugin
